## Known Limitations

### Quiet Hours

- Only one time range per settings (not multiple ranges)
- Uses system time (no automatic timezone adjustments)
- Hours only (can't set 22:30, just 22)
- Respects local system time, not UTC

### General

- Cannot interact with multi-step Claude features
- Extended thinking is auto-disabled
- Requires Claude.ai to be accessible
- Works only on chromium-based browsers
